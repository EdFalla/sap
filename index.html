¿Qué es ABAP?

SAP, como líder en soluciones empresariales, ofrece una amplia gama de herramientas y tecnologías de programación para personalizar, extender y desarrollar aplicaciones. Este ecosistema de herramientas permite a los desarrolladores abordar diferentes necesidades, desde desarrollos tradicionales on-premise hasta soluciones avanzadas en la nube. En esta presentación exploraremos las principales herramientas de desarrollo:

  • SAP Business Technology Platform (BTP).
  • Nuevo ABAP y ADT.
  • RAP (Restful Application Programming Model).

Herramientas de Desarrollo

  • ABAP Workbench: Un conjunto de herramientas integrado en SAP GUI.
  • ADT (ABAP Development Tools): Desarrollo moderno en Eclipse.
  • SE80: Herramienta tradicional para gestionar objetos ABAP.

Sintaxis del Lenguaje

DATA: my_variable TYPE string.
WRITE: 'Hola, Mundo!'.
LOOP AT lt_table INTO wa_table.
  WRITE: / wa_table-field.
ENDLOOP.

Ejemplos de Código

Hola Mundo

DATA: greeting TYPE string VALUE 'Hola, Mundo!'.
WRITE: / greeting.

Recorrido de Tabla Interna

DATA: lt_table TYPE TABLE OF string,
      wa_table TYPE string.
APPEND 'Elemento 1' TO lt_table.
APPEND 'Elemento 2' TO lt_table.
LOOP AT lt_table INTO wa_table.
  WRITE: / wa_table.
ENDLOOP.
